## What You'll Build

A Snake game where:
- Your snake is a **Data Stream** navigating a network grid
- **Red threat nodes** spawn (Phishing, Ransomware, Prompt Injection...)
- **Green defense nodes** spawn paired with each threat
- Collecting the defense pauses the game and shows a **lesson card**
- Hitting the threat triggers a **risk briefing**

## You Need

- Laptop with Code Editor eg. [VS Code](https://code.visualstudio.com), Antigravity, Cursor, etc.
- AI Coding Agents extension eg. Copilot, Claude Code, etc.
- A browser
